Privacy Policy — Kurohyo

最終更新日: 2026年7月7日

本ポリシーは、アプリケーション「Kurohyo」（以下「本アプリ」）の運営者（以下「当方」）が、本アプリの提供にあたって取り扱う情報について定めるものです。本アプリは、当方が権利者の許諾のもとで運用する切り抜きチャンネルの動画を、当方自身が管理するソーシャルメディアアカウント（YouTube、TikTok、Instagram、X 等）へ投稿するために使用する、当方の内部運用ツールです。

1. 取得する情報

本アプリは、一般利用者から個人情報を収集しません。本アプリが取り扱うのは、当方自身のアカウントに投稿を行うために必要な、以下の情報に限られます。

  • 各プラットフォームが発行するアクセストークン（当方アカウントへ投稿する権限）
  • 当方が作成・アップロードする動画ファイルおよびその説明文・タイトル
  • 投稿の成否を確認するための、投稿ID等のメタデータ

2. 情報の利用目的

取得した情報は、当方自身のアカウントへ動画を投稿・管理する目的にのみ利用します。第三者への提供、広告目的での利用、プロファイリングは一切行いません。

3. TikTok から取得する情報の取り扱い

本アプリは TikTok Content Posting API を利用します。当該 API を通じて取得するアクセストークンは、当方アカウントへ動画を投稿する目的にのみ使用し、暗号化された環境変数として安全に保管します。TikTok から取得したいかなる情報も、TikTok の開発者利用規約およびブランドガイドラインに従って取り扱います。

4. 情報の保管と安全管理

アクセストークン等の認証情報は、外部に公開されないローカル環境の設定ファイル（.env 等）に限定して保管し、第三者がアクセスできない状態で管理します。

5. 第三者提供

当方は、法令に基づく場合を除き、取得した情報を第三者に提供しません。

6. 情報の削除

各プラットフォームの連携解除により、対応するアクセストークンは失効し、当方はこれを速やかに破棄します。削除に関するお問い合わせは下記までご連絡ください。

7. 改定

本ポリシーは、必要に応じて改定されることがあります。重要な変更を行う場合は、本ページ上で告知します。

8. お問い合わせ

本ポリシーに関するお問い合わせは、diamondmovie.office@gmail.com までご連絡ください。
